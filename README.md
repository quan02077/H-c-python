# 🐍 Học Python

Repository lưu trữ các bài tập và dự án học tập Python.

## 📖 Nội dung

- Các bài tập cơ bản về Python
- Xử lý dữ liệu với Python
- Ứng dụng thực hành các khái niệm lập trình
- Các thư viện phổ biến

## 🛠️ Công nghệ sử dụng

- **Python 3**
- **NumPy, Pandas** (nếu có)
- **Matplotlib** (nếu có)

## 🚀 Cách sử dụng

1. Clone repository
2. Cài đặt Python 3
3. Chạy file .py:
   ```bash
   python filename.py
   ```

## 📝 Ghi chú

Repository được tạo nhằm mục đích ghi nhận quá trình học Python và ôn luyện kỹ năng lập trình.
